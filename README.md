# DeepFakeDetector
Esta aplicacion fue desarrollada mediante el entrenamiento de un modelo de inteligencia artificial para la clasificación de fotos llamado Teachable Machine.
El modelo se usa mediante el paquete ```@sashido/teachablemachine-node``` y se modificó para el clasificación directa de imagenes.
# Tecnologías
* Angular 17
* ExpressJS
* TypeScript
